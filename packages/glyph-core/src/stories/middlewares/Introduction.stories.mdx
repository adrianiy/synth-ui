###### COMPOSER
# Introduction

A colleciont of middlewares to use in your backends. 

This middlewares are used inside [composer](?path=/docs/composer-introduction--introduction), but you can use them as standalone middlewares. 

#### Implementation example

```typescript
const middleware = (input) => {
  return async (ctx, next) => {
    await next();
  }
}
```
