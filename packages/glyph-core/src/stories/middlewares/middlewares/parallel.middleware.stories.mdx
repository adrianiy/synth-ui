###### Middlewares
# Parallel

Execute scripts **parallalized**.

This is util when the executions are not output dependant like grouping data in different state keys

> Accepts an array of properties

### Example

```typescript
import { parallel } from 'glyph-core-poc';

service()
  .pipe(
    parallel(middleware(), middleware())
  )
```
