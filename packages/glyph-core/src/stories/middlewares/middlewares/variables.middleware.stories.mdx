###### Middlewares
# SetVariables

Save object in `ctx.state`.

These variables will be accesible through all request context.

### Example

```typescript
import { setVariables } from 'glyph-core';

service()
  .pipe(
    setVariables({ test: 'documentation', testb: 'glyph'})
  )
```
