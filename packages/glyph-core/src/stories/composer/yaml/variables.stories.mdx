###### Configuration
# variables

Save object in `ctx.state`.

These variables will be accesible through all request context.

### Example

```yaml
variables:
    testVariable: documentation
    testVariable2: 999

# ctx.state -> { testVariable: 'documentation', testVariable2: 999 }
```
