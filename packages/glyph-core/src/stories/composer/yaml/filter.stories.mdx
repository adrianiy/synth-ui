###### Configuration
# filter

Filter data in `data` key using some requirement function or keys.

If keys are setted we'll remove row keys depending on the configuration value.

#### Parameters

| NAME | TYPE | DESCRIPTION | DEFAULT VALUE|
|:-----|:-----:|:---------|------:|
|data|String|Key of `ctx.state` where data is stored | data |
|store|String|Key of `ctx.state` where data will be stored | data |
|function|Method or key name|null|
|required|String[]|Array of keys that must be informed|null|
|keys|{ preserve, remove } | parse each row to filter keys by it's value |null|

### Example 

```yaml
name: filter row without name
filter:
  required:
    - name
  keys:
    preserve:
      - name
      - cod_country
```

```yaml
name: filter using function
filter:
  function: row => row.value > 100
```
