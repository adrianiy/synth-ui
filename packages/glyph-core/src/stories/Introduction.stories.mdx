###### INTRODUCTION
# GlyphUI-Core

This library is the core of GlyphUI components library, implementing:

- Data interfaces.
- State modification functions, vanillajs methods based on React-Redux.
- Middlewares to use on frontend backends.
- Composer to create custom middlewares pipes using yml configuration files


## Install

```bash
# yarn
yarn add glyph-core

# npm
npm i glyph-core
```
