# SynthUI-React-Components
#### React library implemented over webComponents

This library allow to share logic across Drive React applications. Developers do not need to care about how components interact with application state, or how they
transform raw data into usable components.

## Use

To use this React library simply import components where you want to use them.

```javascript
import { SynthReactList } from 'synth-react';

...

    return (
        <div className="App">
            <div className="list">
                <SynthReactList title={'List'} limit={3} data={data} fieldsConfig={fieldsConfig} />
            </div>
        </div>
    );
...
```

## Local Development

Execute

### Using docs

```bash
  yarn start:docs
```
or 

```bash
lerna run start:docs --scope synth-react --stream
```

You'll have react docs available on port *6009*.

### Using demo playground

From root directory.

```bash
  yarn start:front 
```
or 

```bash
lerna run srtart --scope synth-react --scope react-demo
```
