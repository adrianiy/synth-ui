# GlyphUI-Angular-Components
#### Angular library implemented over web components

<Story id="introduction--introduction"/>

This library allow to share logic across Drive Angular applications. Developers do not need to care about how components interact with application state, or how they
transform raw data into usable components.

## Install

```bash
# yarn
yarn add glyph-angular

# npm
npm i glyph-angular
```

## Use

To use it as Angular Component library, simply import modules in app.module so you can use in all your app.

exported modules:

- GlyphAngularComponents
- GlyphAngularServices

> To inject services at root level use `forRoot()` as shown below

```javascript
import { GlyphAngularComponents, GlyphAngularServices } from 'glyph-angular-poc';

@NgModule({
    declarations: [ ... ],
    imports: [
        ...
        GlyphAngularComponents,
        GlyphAngularServices.forRoot(),
        ...
    ],
    providers: [ ... ],
    bootstrap: [ ... ],
})
export class AppModule {}
```

### Custom services implementations

The second argument of `GlyphAngularServices.forRoot()` is an object of classes to inject your custom services implementations in root.

You can create a custom class that extends FilterService

```javascript
import { Injectable } from '@angular/core';
import { FilterService } from 'glyph-angular-poc'

@Injectable()
export class CustomFiltersServie extends FilterService {
  constructor(...) {
    super(...)
  }

  // this is an extended method
  initializeFilters(...) {
    super.initializeFilters(...);
    // Custom filters initialization logic
  }

  // this is a new method
  customMethod() {
    // this is a custom method
  }
}
```

Now you can pass an object to `forRoot` informed with your custom implementations

```javascript
import { GlyphComponentsModule, GlyphServicesModule } from 'glyph-angular-poc';

@NgModule({
    declarations: [ ... ],
    imports: [
        ...
        GlyphAngularServices.forRoot({ glyphFilterService: CustomFiltersServie }),
        ...
    ],
    providers: [ ... ],
    bootstrap: [ ... ],
})
export class AppModule {}
```


## Local Development

Execute

### Using docs

```bash
yarn angular start:docs
```

You'll have angular docs available on port *6008*.

### Using demo playground

```bash
yarn angular start
```

You'll have an angular application running on port *4200*.
