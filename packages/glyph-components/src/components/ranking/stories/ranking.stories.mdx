###### glyph-ranking
# Ranking

This component can be rendered as a single ranking (grid) or a ranking with three columns (woman, man, kid).

The **articles** can have a propert `image` or `plain_image` with a full url, or be configured using `parseImageUrl` callback, wich
should receive `image` or `plain_image` property and return an url string.

> Single section header is customizable with `rankingHeader` prop
