###### glyph-scroll
# Scroll

Scrollable component.

Renders minimalistic scrollbar either for y-scroll or x-scroll.

You can use `width` and `height` element properties or pass a `containerClass` which sets `--scrollbar-height` and `--scrollbar-width`
