###### glyph-flex
# Flex

A helper component to distribute content in your application.

Creates a flex component (with column direction by default) that uses some boolean input options to distribute content inside (see arg. table below).

Go to **Play with it!** tab and try some configurations

### Column

Predefined column direction with center and middle alignment.

<Canvas>
    <Story name="Column" args={{ ...defaultArgs, center: true, middle: true }}>{Template.bind({})}</Story>
</Canvas>

### Row

Row direction with spaced and middle alignment.

<Canvas>
    <Story name="Row" args={{ ...defaultArgs, row: true, spaced: true, middle: true }}>{Template.bind({})}</Story>
