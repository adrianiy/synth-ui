# Header

Header component used in Drive applications.

It renders the application brand and application title on left side, and a set of action buttons on the right side.


> These buttons can be enabled on demand. Implemented action buttons are:
>
> | action        | icon      | onClick                                                          |
> |---------------|-----------|------------------------------------------------------------------|
> | search        | magnifier | show a search input                                              |
> | notifications | bell      | show a list of notifications                                     |
> | share         | share     | show a widget to share application link, with or without filters |
> | menu          | apps      | show an applications grid                                        |
> | timeline      | calendar  | shows a timeline sidebar (on the right side)                     |
> | user          | avatar    | shows user menu                                                  |
